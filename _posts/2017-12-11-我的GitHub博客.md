### **GitHub Page**建立个人博客
### GitHub Page
* 功能

允许用户自定义项目首页，即*用户编写*，然后托管在GitHub上的静态网页。
* GitHub Page意图

一个简明易懂的网页，说明**项目(Repository)**如何运行，帮助入门。
* 个人理解
博客网页允许GitHub站内生产网页，也可以自己编写(使用Jekyll)。
